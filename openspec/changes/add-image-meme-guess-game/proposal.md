# Change: 新增看图猜梗游戏

## Why

平台已有「看图猜物品」等玩法，缺少面向梗图/表情包场景的轻量猜题游戏。通过配置多道「图片 + 答案」题目，玩家看图输入答案即可，无需提示开关与联想输入，猜对后可再来一局，适合年会、团建等场景。

## What Changes

- 新增「看图猜梗」游戏：配置人员在设置中配置多道题目，每题包含图片与答案，无提示功能（不提供提示开关）。
- 游玩流程：进入游戏后展示一张图片，玩家在**无联想下拉**的输入框中输入答案；默认不限制每题尝试次数，可在游戏配置页可选配置「最大尝试次数」与**可选倒计时**；猜对后显示成功、**展示答案**，并支持「再来一局」；**次数用尽后也统一展示「再来一局」**。**界面极简，不展示本局统计**。题目选取：每次从**尚未使用过**的题目中**随机**选一道，用光题库后自动重置记忆再选；「已使用」记忆**仅存内存**，刷新即清；某题图片加载失败则**自动跳过**该题。游戏内页头提供**「重置记忆」**按钮；题库仅 1 题时在页面或成功提示中说明。答案匹配**复用**电影猜测等现有工具与规则。**配置变更与进行中题目被删除**：与 settings-admin 一致（进行中不生效，删除题不影响当前局）。
- 扩展设置管理：新增看图猜梗的数据 Schema（图片支持 **URL 与本地上传**，答案 + 可选别名）、独立配置页与数据管理（CRUD、默认/自定义、导入导出，**500 条上限**，与现有游戏一致）；游戏参数含可选倒计时与可选最大尝试次数。**默认 2 条示例题目**（pigGuess.png→二锅头，roubaoGuess.png→书包），开箱可玩；自定义数据参与**数据过期**清除。
- 平台集成：在首页、路由 **`/image-meme-guess`**、设置入口中注册看图猜梗，与现有游戏并列展示；**中英 i18n** 均补充看图猜梗相关文案。

## Impact

- Affected specs:
  - 新增 `image-meme-guess-game` 能力（描述题目展示、纯文本输入、答案匹配、再来一局）。
  - 修改 `settings-admin`（补充看图猜梗数据 Schema：图片 + 答案，无提示开关）。
- Affected code:
  - `src/views/`（新建看图猜梗游戏页，如 `ImageMemeGuess.vue`）。
  - `src/config/games.ts`、`src/router/index.ts`、`src/views/Home.vue`、`src/views/Settings.vue`（注册新游戏与设置卡片）。
  - `src/views/GameSettings/`（新建看图猜梗配置页，如 `ImageMemeSettings.vue`）。
  - `src/schemas/`、`src/data/`、`src/utils/`（Schema、默认数据、存储与匹配逻辑）。
  - `src/locales/`（看图猜梗相关 i18n 键值）。
