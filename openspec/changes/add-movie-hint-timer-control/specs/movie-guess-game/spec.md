## ADDED Requirements

### Requirement: 电影初始提示显示
电影猜测游戏 SHALL 在游戏开始时显示配置的初始提示，帮助玩家缩小猜测范围。

#### Scenario: 显示配置的初始提示
- **GIVEN** 电影配置了 `hint` 字段（如"漫威"）
- **AND** 全局设置中"显示初始提示"开关已开启
- **WHEN** 游戏开始
- **THEN** 在游戏界面显示初始提示区域
- **AND** 提示内容为配置的 `hint` 值

#### Scenario: 全局开关关闭时不显示提示
- **GIVEN** 电影配置了 `hint` 字段
- **AND** 全局设置中"显示初始提示"开关已关闭
- **WHEN** 游戏开始
- **THEN** 不显示初始提示区域

#### Scenario: 电影未配置提示时不显示
- **GIVEN** 电影未配置 `hint` 字段（或为空字符串）
- **AND** 全局设置中"显示初始提示"开关已开启
- **WHEN** 游戏开始
- **THEN** 不显示初始提示区域

#### Scenario: 提示样式与城市游戏一致
- **GIVEN** 电影配置了提示且全局开关开启
- **WHEN** 显示初始提示
- **THEN** 使用与城市猜测游戏相同的UI样式（蓝色背景、灯泡图标）
