# 本地资源服务 - 新手步骤（无需会 GitHub）

按下面顺序做即可，**不需要会写代码或使用 GitHub**。

---

## 第一步：安装 Node.js（只需做一次）

1. 打开浏览器，访问：**https://nodejs.org/**
2. 点击页面上的 **「LTS 推荐」** 绿色按钮，下载安装包。
3. 下载完成后**双击**安装包，一路点 **「下一步」**，最后点 **「完成」**。
4. 装好后**不用**打开任何东西，进行第二步即可。

---

## 第二步：拿到项目文件夹

1. 打开浏览器，访问：**https://github.com/JustinBIBERRR/gameRepo**
2. 点击绿色的 **「Code」** 按钮，再点击 **「Download ZIP」**，下载压缩包。
3. 打开**下载**文件夹，找到刚下的 `gameRepo-main.zip`，**右键 → 解压到当前文件夹**（或解压到桌面、D 盘等任意位置）。
4. 解压后会有一个文件夹，例如 **`gameRepo-main`**，记住它在哪里（例如 `E:\gameRepo-main` 或 `桌面\gameRepo-main`）。后面说的「项目文件夹」就是指这个。

---

## 第三步：准备好你的资源文件夹

1. 把你的**视频文件**（如 `mry.mp4`）和**音频文件**（如 `.mp3`）都放进**同一个文件夹**里。
2. 这个文件夹可以是任意位置，例如：
   - `C:\Users\你的用户名\Videos\gameRepo`
   - 或 `D:\我的视频`
   - 或 `E:\gameRepo`
3. **记住这个文件夹的完整路径**（在资源管理器顶栏可以看到或复制），例如：`C:\Users\w00813752\Videos\gameRepo`。后面会用到。

---

## 第四步：在项目里填上你的资源路径

1. 打开 **项目文件夹**（第二步里的 `gameRepo-main`）。
2. 再打开里面的 **`scripts`** 文件夹。
3. 找到 **`resource-server.config.txt`**，**右键 → 用记事本打开**。
4. 在文件里**新起一行**（不要用 `#` 开头），**只写你的资源文件夹路径**，例如：
   ```text
   C:\Users\w00813752\Videos\gameRepo
   ```
   （把你自己的路径替换进去，不要有引号，不要有空格在路径前后。）
5. 点 **「文件」→「保存」**，关掉记事本。

---

## 第五步：双击启动（以后每次用都这样）

1. 还是在 **项目文件夹 → `scripts`** 里。
2. 找到 **「启动本地资源服务.bat」**，**双击**它。
3. 会弹出一个**黑色窗口**，不要关它；浏览器会自动打开一个页面，上面会列出你的视频/音频文件名。
4. 在游戏平台里配置电影或歌曲时，把地址填成：
   - **`http://127.0.0.1:8080/文件名`**
   - 例如：`http://127.0.0.1:8080/mry.mp4`
5. 用完后**关掉黑色窗口**，服务就停了；下次再用就再**双击一次**「启动本地资源服务.bat」即可。

---

## 常见问题

| 情况 | 处理 |
|------|------|
| 双击 .bat 后闪退或提示「未检测到 Node.js」 | 回到**第一步**，确认 Node.js 已安装，并**重启电脑**后再试。 |
| 黑色窗口里写「目录不存在」 | 检查**第四步**里填的路径是否和**第三步**的资源文件夹路径**完全一致**（可到资源管理器里复制路径粘贴进去）。 |
| 浏览器里是 Not Found | 确保**第五步**里没有关掉黑色窗口；在浏览器地址栏输入 `http://127.0.0.1:8080/` 再回车。 |
| 游戏里填了地址仍播不出 | 确认黑色窗口一直在运行；地址要带 `http://`，且文件名和资源文件夹里的**完全一致**（含扩展名，如 `.mp4`）。 |

---

**总结：** 安装 Node.js → 下载并解压项目 → 在 `scripts\resource-server.config.txt` 里写上你的资源路径并保存 → 双击 `scripts\启动本地资源服务.bat` → 在游戏里用 `http://127.0.0.1:8080/文件名` 即可。
